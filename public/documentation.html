Sci-Fi-App Documentation

Purpose

The purpose of this project is to build a complete web application (client-side and server-side) built using full-stack JavaScript technologies. The project will demonstrate full-stack JavaScript development, including APIs, web server frameworks, databases, business logic, authentication, data security, and more. The tech stack used is known as the MERN (MongoDB, Express, React, and Node.js).

Current State

The server is set up with using node.js and express. The website can be locally hosted. There are three endpoints that can be reached and handled, that is "/", "movies" and "documentation". Requests are logged by middleware using morgan.

APIs

Return all movies

Return a list of ALL movies as a JSON file.

GET /movies

Return a single entry

Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user

GET /movies/:title

Return data about a genre

Return data about a genre (description) by name/title (e.g., “Thriller”)

GET /movies/genres/:name

Return data about a director

Return data about a director (bio, birth year, death year) by name

GET /movies/director

Register new user

Allow new users to register

POST /users/register/:username

Update user name

Allow users to update their user info (username)

PUT /users/:name/:newusername

Add movie to favorites

Allow users to add a movie to their list of favorites (showing only a text that a movie has been added)

POST /users/:favoritemovie

Remove a movie from favorites

Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed)

DELETE /users/:removefavmovie

Deregister user

Allow existing users to deregister (showing only a text that a user email has been removed)

DELETE /users/:name
