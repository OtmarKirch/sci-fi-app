Sci-Fi-App Documentation

Purpose

The purpose of this project is to build a complete web application (client-side and server-side) built using full-stack JavaScript technologies. The project will demonstrate full-stack JavaScript development, including APIs, web server frameworks, databases, business logic, authentication, data security, and more. The tech stack used is known as the MERN (MongoDB, Express, React, and Node.js).

Current State

The server is set up with using node.js and express. The website can be locally hosted. Several endpoints have been implemented for retrieving movie data, registering users and adding favorite movies. Requests are logged by middleware using morgan.

Register new user

Allows new users to register.

Add a body in JSON format like this: {"name": "mike", "email": "mike@mike.com"}. This example will register a new user with the username "mike" and the email address "mike@mike.com

POST /users/register

Update user name

Allow users to update their user info (username)

PUT /users/{username}

Deregister user

Allow existing users to deregister their email address.

Add a body in JSON format like this: {"name": "mike"}. This example will delete Mike's email address in the registry.

DELETE /users/delete

Add movie to favorites

Allows users to add a movie to their list of favorites.

POST /users/{username}/{favoritemovie}

Remove a movie from favorites

Allows users to remove a movie from their list of favorites.

DELETE /users/{username}/{favoritemovie}
